# 조건문

### 기본 조건문 사용방법

```python
if 조건문 1:
    code
elif 조건문 2:
    code
else:
    code
```

### 비교 연산자

- x==y / x!=y
- x<y / x>y
- x>=y / x<=y

### 논리 연산자

- 파이썬에서는 문자로 표현한다
- && = and
- || = or
- ! = not

### 기타 연산자

- X in 리스트(문자열)
- X not in 리스트(문자열)

### 조건부 표현식

```python
score = 85
result = "Success" if score >=80 else "Fail"
```

# 반복문

### while

```python
i=0
while i<=5:
    i++
```

### for

```python
for i in range(0,5):
    print(i)

list = [1,2,3,5]
for i in list:
    print(i)
```
